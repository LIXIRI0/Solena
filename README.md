# Solena
A tiny GPT-style language model implemented from scratch in PyTorch.  
Built for learning, experimenting, and deploying small-scale transformers.

## Features
- Character-level tokenizer
- Configurable model size
- GPU/CPU compatible
- Training script + dataset loader
- Clean modular structure

## Roadmap
- Add attention module
- Add training checkpoints
- Add text generation
- Add optimizer + LR scheduling
